# Sommel

Sommel is a Rails MVC web-application to log your favorite wines, add reviews/tasting-notes and discover what bottle to check out next.

## Installation

If you want to demo Sommel, feel free to fork and clone this repo.

$ bundle install

$ rails db:migrate

$ rails db:seed

$ rails server
